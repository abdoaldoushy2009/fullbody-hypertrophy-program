# 💪 Full Body Hypertrophy Program

A science-based 3-day full body training program built as a single self-contained HTML file — no frameworks, no dependencies, no server required.

## 🚀 Live Demo

Open `index.html` in any browser. No internet connection required after download.

## ✨ Features

- **3-day full body split (A/B/C)** with 2× weekly frequency per muscle group
- **Volume within MEV–MAV range** (10–14 sets/muscle/week)
- **Double progression model** — reps first, then weight (linear overload)
- **Macro & Nutrition Calculator** powered by:
  - Mifflin–St Jeor formula for BMR
  - Harris-Benedict activity multipliers for TDEE
  - Science-based macro split (Protein · Carbs · Fat)
- **4 Languages** — English · Arabic (RTL) · Spanish · German
- **Offline-ready** — download and run with zero internet
- **PWA installable** — add to home screen on mobile/desktop

| Principle | Detail |
|-----------|--------|
