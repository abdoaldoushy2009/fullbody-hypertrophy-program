A science-based 3-day full body training program built as a 
single self-contained HTML file — no frameworks, no dependencies, 
no server required.

Features:
- 3-day full body split (A/B/C) with 2× weekly frequency per muscle group
- Volume within MEV–MAV range (10–14 sets/muscle/week)
- Double progression model (reps → weight linear overload)
- Built-in Macro & Nutrition Calculator (Mifflin–St Jeor BMR + Harris-Benedict activity multipliers)
- Calculates daily calories, protein, carbs, and fat based on weight, height, age & goal
- 4 languages: English, Arabic (RTL), Spanish, German
- Download for offline use — works with no internet connection
- Installable as a PWA on mobile and desktop

Stack: Vanilla HTML · CSS · JavaScript — zero dependencies.
